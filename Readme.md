# Jungle Runner

## Overview
Jungle Runner is a 2D endless runner game inspired by the classic T-Rex runner. The player runs automatically through a jungle-themed environment and must jump over incoming obstacles to survive as long as possible. The score increases over time, and the game becomes more difficult as obstacle speed increases.

## Game Loop
1. The player runs automatically.
2. Obstacles appear from the right side of the screen.
3. The player jumps to avoid obstacles.
4. The score increases while the player survives.
5. The game speed gradually increases.
6. The game ends when the player hits an obstacle.
